About

私について

愛知県を拠点にWebコーディングを行っています。工業高校を卒業後、製造業で設備の保守保全に約8年、その後自動車整備士として勤務し、2025年からWeb制作の学習を始めました。

前職で培った「図面の読みやすさ」「直しやすい構造」の感覚を、HTMLとCSSの設計に活かしています。派手さよりも、半年後・一年後に誰が見ても迷わないコードを書くことを大切にしています。

Skills

使える技術

  • HTML

    セマンティックなマークアップ。見出し階層・landmark・ARIA属性の扱いを意識し、SEOとアクセシビリティに配慮した構造で書きます。

  • CSS / Sass

    FLOCSS + BEM によるCSS設計。Foundation / Layout / Object の階層で責務を分け、長期的にメンテしやすいスタイルシートを構築します。

  • JavaScript

    バニラJSで動的挙動を実装。IntersectionObserver / DOM API を中心に、過剰なライブラリ依存を避けたシンプルな設計を好みます。

  • Responsive Design

    モバイルファーストで SP / TB / PC の3段階に対応。clamp() を活用し、ブレイクポイント間でも破綻しない流動的な設計を行います。

  • Git / GitHub

    基本的なブランチ運用・Pull Request・コードレビュー対応ができます。コミットメッセージの粒度と意図の明示を意識します。

  • Accessibility

    キーボード操作・スクリーンリーダー・prefers-reduced-motion など、利用者の状況に応じたフォールバックを考慮した実装を行います。
